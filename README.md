# Python-Tools Framework

This repository serves as a centralized management hub for various automation modules. Each subdirectory contains specialized tools and its own technical documentation.

## Projekt-Struktur / Module
| Modul | Kurzbeschreibung | Dokumentation |
| :--- | :--- | :--- |
| **Automation-Helpers** | All systems are managed under ISO/IEC 26514 compliant documentation standards. | [View Docs](./Automation-Helpers/) |
| **Network-Scanner** | This module contains a simple TCP port scanner written in Python. | [View Docs](./Network-Scanner/) |
| **Web-Crawler** | This module provides an interactive web crawler designed to extract article-like content from websites, offering features for filtering, blacklisting, and generating live HTML reports. | [View Docs](./Web-Crawler/) |
